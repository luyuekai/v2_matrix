HDFS文件资源管理器给使用者提供浏览和操作Hadoop Distributed File System (HDFS)中文件和目录的便捷能力，它可以帮助你：
  • 对其关心的大数据分析文件进行管理和查看。
  • 无须复杂的命令行交互，使用它就如同像童年时使用windows资源管理器一样简单。
除此之外，我们还提供了很多额外的功能与特性，比如：
  • 下载、重命名、移动、复制、删除...
  • 迭代更改权限。
  • 迭代更改Ownership。
  • 灵活的属性排序(基于文件名、文件大小、修改日期...)。
  • 查看gzip/gz文件的内容。
  • 查看大文件内容。
  • 基于权限的文件上传管理。
  • 大文件的格式验证。
尝试着使用它，并依赖上它，是我们的目标与改进方向
元数据管理器和数据资源管理器给使用者浏览和管理不同database的能力，比如：
  • 浏览Hive数据库中的table、partitions、buckets，创建表，并管理它...
  • 对RDMS型数据库中的table视图化管理
  • 定制化元数据，使其满足用户统计功能等使用
Job管理器给与使用者检查运行在Hadoop集群上面的Mapreduce jobs的能力，对其job和task进行监测。
  • 首先看到的是处于最外层/顶端的job列表
  • 对于关心的job，可以下探/查看它关联的task列表
  • 对于关心的task，可以查看它关联的attempts和其每个attempt的属性，比如状态、起止时间、输出大小
  • 对于失败的job，可以进行troubleshoot

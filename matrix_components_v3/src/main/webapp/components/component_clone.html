Clone Design

Clone 功能给予开发人员快速clone组件的能力

Clone相关api

  • 引入generic_workbench.js

  • api: 

    描述：复制一个已有的dom元素
clone_component_v2(div_id,clone_div_id)
+  参数 div_id:准备被克隆的组件id，必填
+  参数 clone_div_id: 克隆的新组件id，选填，如果不填，系统默认生成唯一标示
+  返回值: 返回新克隆的dom，但此时注意，此元素没有被render

  • api: 

    描述：复制一个已有的dom元素，并将其放置到dom中的另一个元素内
clone_component(destination_parent_div_id,div_id,clone_div_id)
+  参数 destination_parent_div_id:克隆后的组件被render到的dom元素id，必填
+  参数 div_id:准备被克隆的组件id，必填
+  参数 clone_div_id: 克隆的新组件id，选填，如果不填，系统默认生成唯一标示
+  返回值: 返回新克隆的组件id

  • api:

    描述：将一个已有的dom元素放置到workbench
add_content_div(content,x,y,x_width,y_height)
+ 参数 content： 准备copy的dom元素
+ 参数 x: workbench的x
+ 参数 y: workbench的y
+ 参数 x_width: workbench上content占用的宽度，最大为12
+ 参数 y_width: workbench上content占用的高度，最大为12

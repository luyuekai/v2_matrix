自定义Client端CSV下载控件

为什么要制作自定义Client端CSV下载控件

系统往往需要数据下载到本地的功能，一般而言，数据在从server端取得返回给client端渲染后，比如显示成表格后，
在用户想要到出成csv文件时，如果数据已经完全取回，没有必要再次调用server端代码实现取数据+下载的功能，提供更好的性能和扩展性

Dependencies:

JQuery

File locations:

assets/self-owned/js/self-csv.js

Download csv file without server. Just pass data array and choose the file name:

var header = ['header1','header2','header3'];
var data = [ [1,2,3],['data1','data2','data3'] ];
var csv_name = 'notebook_download_data.csv';
var csv_content = [];
csv_content.push(header);
csv_content = csv_content.concat(data);
csvGenerator = new CsvGenerator(csv_content,csv_name);
csvGenerator.download(true);

If you want to create the element:

csvGenerator = new CsvGenerator(csv_content,csv_name);
csvGenerator.getLinkElement();

If you want just to get href attribute of the link:

csvGenerator = new CsvGenerator(csv_content,csv_name);
csvGenerator.getLinkElement(true);

Formatting options

// Semi colon separated
csvGenerator = new CsvGenerator(csv_content,csv_name, ';');
// Semi colon separated, all cells quoted
csvGenerator = new CsvGenerator(csv_content,csv_name, ';', true);

Table由Header和Data构成
Header为一维数组，数组中的元素按顺序代表表头名，在教程中，它为["姓名", "年龄"]
Data为二维数组，每一个内层数组代表一行数据，在教程中，它为[["Alick", "56"],["Jason", "55"]...]
 信息! 请牢记：上面的两个数据结构是构造任何的Table基础 
1,准备一个可供Table显示的Div
                      
                        <div id="my_first_table_div"></div>
                      
2,准备一个可供Table显示的数据源
                      
                        var ds = {
                            'type': 'static',
                            'header': ["姓名", "年龄"],
                            'result': [
                              ["Alick", "56"],
                              ["Jason", "55"],
                              ["Tom", "28"],
                              ["Jerry", "16"],
                              ["Aaron", "35"],
                              ["Daniel", "53"],
                              ["Life", "26"],
                              ["Chapter", "75"],
                              ["Frank", "28"]
                            ],
                            'isDisplayPager':true,
                            'pageMaxSize':5
                        };
                      
3,执行Matrix创建Table函数
                      
                        create_table_template('my_first_table_div',new MatrixTableTemplate(),ds);
                      
这样你就创建了一个Matrix Table，是不是很简单？尝试着做你自己的第一个Table吧。

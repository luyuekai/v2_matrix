request_remote(url,handler,option_param.option_type,option_need_wrap,option_wrap_data,option_respones_error_handler,option_service_error_handler)

  • 作用：访问数据库等远程数据源
    • url（必填）：请求地址
    • handler（必填）：处理请求到的数据的函数
    • option_param（选填）：发送的请求的参数
    • option_type（选填）：请求方式，默认为POST
    • option_need_wrap（选填）：请求是否包含额外数据
    • option_wrap_data（选填）：请求的额外数据，即可与请求数据进行关联处理的属性
    • option_responses_error_handler（选填）：请求错误返回发送语句，通过$.subscribe()进行捕获
    • option_service_error_handler（选填）：请求发生服务器错误返回语句，通过$.subscribe()进行捕获
  • 使用：Matrix_Util.request_remote(url,handler,...)
  • 举例：在console中输入 MatrixUtil.requestremote(MatrixUtil.getprojectpath() + '/api/mock/post/success', MatrixUtil.print_handler); 输出为
{
  "hasError": false,
  "errorMessage": null,
  "statusCode": 200,
  "status": null,
  "result": null,
  "totalCounts": null,
  "pageMaxSize": null,
  "currentPageNumber": null,
  "currentPageSize": null
}

json2table(inputData)

  • 作用：将JSON格式数据转化为适应table格式的数据
    • 返回result：
      • header ： 一维数组
      • result ： 二维或多维数组
    • inputData：JSON格式数据
  • 使用：Matrix_Util.json2table(inputData)
  • 举例：
//在console中输入
var inputData =   [
     {"id": 1,"name": "bill","sex": "male","age": 32,"email": "bill@matrix.com","phone": 13912738475},
     {"id": 2,"name": "amanda","sex": "female","age": 32,"email": "amanda@matrix.com","phone": 13912738475},
     {"id": 3,"name": "carina","sex": "female","age": 32,"email": "carina@matrix.com","phone": 13912738475},
     {"id": 4,"name": "tom","sex": "male","age": 32,"email": "tom@matrix.com","phone": 13912738475}
   ]
Matrix_Util.json2table(inputData)

//在console中返回
{
  header: ["id", "name", "sex", "age", "email", "phone"]
  result: [
    [1, "bill", "male", 32, "bill@matrix.com", 13912738475],
    [2, "amanda", "female", 32, "amanda@matrix.com", 13912738475],
    [3, "carina", "female", 32, "carina@matrix.com", 13912738475],
    [4, "tom", "male", 32, "tom@matrix.com", 13912738475]
  ]
}

  • 作用：获取JSON数组的所有key属性列表
    • inputDataArray：JSON数组
